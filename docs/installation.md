# Installation

## Stable release

To install ANDALUS, run this command in your terminal:

```sh
uv add andalus
```

Or if you prefer to use `pip`:

```sh
pip install andalus
```

## From source

The source files for ANDALUS can be downloaded from the [Github repo](https://github.com/daan1392/andalus).

You can either clone the public repository:

```sh
git clone https://github.com/daan1392/andalus
```

Or download the [tarball](https://github.com/daan1392/andalus/tarball/main):

```sh
curl -OJL https://github.com/daan1392/andalus/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd andalus
uv sync
```
